============

Prime Number Generator using JADE cluster

To run the project:

    export JAVA_HOME=/path/to/jdk8
	
	git clone https://github.com/suen/PrimeCluster.git
	cd PrimeCluster 

build the system with: 

	ant

and run with

	ant run
